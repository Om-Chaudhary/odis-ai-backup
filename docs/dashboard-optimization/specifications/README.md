## Specification Format

Each specification includes:

- **Purpose** - What it does and why
- **Props/Interface** - TypeScript interfaces
- **Visual Design** - Mockups and layouts
- **Behavior** - Interactions and states
- **Styling** - Colors, spacing, responsive design
- **Integration** - Where and how it's used
