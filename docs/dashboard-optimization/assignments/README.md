## Assignment Status Legend

- **Ready** - Fully specified, ready to start
- **In Progress** - Currently being worked on
- **Blocked** - Waiting on dependencies
- **Complete** - Finished and merged

## How to Claim an Assignment

1. Check if assignment has dependencies
2. Ensure dependencies are complete (or work in parallel if allowed)
3. Review the full assignment document
4. Start implementation
5. Update status when complete

---

**Note:** Dependencies are minimal. Most assignments can start immediately.
