# Color Palette

## Primary Colors

- **Teal:** `#31aba3`
- **Dark Teal:** `#2a9a92`
- **Light Teal:** `#31aba3/10` (10% opacity)

## Neutral Colors

- **Slate:** slate-50 to slate-900 (Tailwind)
- **White:** `#ffffff`
- **Borders:** `border-slate-200`

## SOAP Note Section Colors

- **Subjective:** `bg-blue-50` border `border-blue-200`
- **Objective:** `bg-teal-50` border `border-teal-200`
- **Assessment:** `bg-purple-50` border `border-purple-200`
- **Plan:** `bg-emerald-50` border `border-emerald-200`

## Usage

- Use teal for primary actions and highlights
- Use slate for text and borders
- Use section colors for SOAP note components
- Maintain consistency across all components
