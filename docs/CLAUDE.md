# docs/CLAUDE.md

This is the Docusaurus documentation site (`apps/docs/`).

## Commands

- `nx serve docs` -- Start docs dev server
- `nx build docs` -- Build static documentation

## Conventions

- UPPER_SNAKE_CASE for guide filenames (e.g., `TESTING_STRATEGY.md`)
- kebab-case for feature docs (e.g., `react-email-templates.md`)
- Link to source code rather than duplicating it in docs
- Update `docs/README.md` when adding or moving documentation
