# Component Development Commands

This directory contains commands and workflows for developing React components in this project.

## Overview

Component development workflows include:

- Component scaffolding
- Visual component verification
- Styling and responsive checks
- Testing component in isolation
- Integration with existing components

## Quick Start

### Create New Component

1. Use shadcn/ui CLI for base components
2. Create component file with proper structure
3. Add to component directory
4. Test visually in browser
5. Add to storybook (if applicable)

### Test Component Visually

1. Navigate to component page/story
2. Test different states
3. Test responsive behavior
4. Verify styling
5. Check accessibility

## Common Workflows

### Component Scaffolding

- Create component file
- Add TypeScript types
- Add Tailwind styling
- Add to component index

### Visual Verification

- Test in browser
- Check responsive breakpoints
- Verify styling consistency
- Test interactive states

### Integration Testing

- Test with parent components
- Verify prop passing
- Check state management
- Test error boundaries
