# Debugging Commands

This directory contains commands and workflows for debugging issues in this project.

## Overview

Debugging workflows include:

- Error investigation
- Performance debugging
- Network request analysis
- Console error checking
- State debugging

## Quick Start

### Debug an Issue

1. Reproduce the issue
2. Check console for errors
3. Monitor network requests
4. Inspect component state
5. Document findings

### Performance Debugging

1. Measure load times
2. Check network requests
3. Analyze render performance
4. Identify bottlenecks

### Error Investigation
