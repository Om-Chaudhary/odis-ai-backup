## Best Practices

1. **Visible Options** - All options visible at once
2. **Single Click** - No dropdown interaction
3. **Clear Active State** - Visual distinction obvious
4. **Keyboard Accessible** - Arrow keys to navigate
5. **Responsive** - Stack on mobile, horizontal on desktop
