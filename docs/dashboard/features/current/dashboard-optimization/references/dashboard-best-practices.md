# Dashboard Best Practices - Research

## Key Principles

1. **Prioritize Actionable Metrics** - Focus on data that drives decisions
2. **Condensed, Expandable Components** - Keep initial view clean
3. **Clear Visual Hierarchy** - Most important info above the fold
4. **Workflow Metrics** - Show completion rates, bottlenecks
5. **Context and Comparisons** - Include trends and benchmarks

## Actionable Metrics

Focus on:

- Items needing attention
- Completion rates
- Workflow health
- Bottlenecks

Avoid:

- Static counts without context
- Metrics that don't drive decisions
- Overwhelming information density

## Layout Hierarchy

**Above the Fold:**

- Key metrics
- Action items
- Trends/charts

**Below the Fold:**

- Detailed lists
- Activity timelines
- Historical data

## Component Patterns

- Button groups for filters
- Expandable sections for details
- Color-coded sections
- Progress indicators
- Action buttons for navigation
