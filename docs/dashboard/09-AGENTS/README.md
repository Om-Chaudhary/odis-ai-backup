# Agent Documentation

This directory contains documentation for AI agents working on dashboard development.

## Agent Resources

- **[Progress Tracker](./AGENT_PROGRESS_TRACKER.md)** - Real-time tracking of agent progress on dashboard assignments
- **[Quick Start Guide](./AGENT_QUICK_START.md)** - Quick start guide for agents working on dashboard
- **[Assignment Template](./AGENT_ASSIGNMENT_TEMPLATE.md)** - Template for creating new agent assignments

## Related Documentation

- [Implementation Assignments](../05-FEATURES/IMPLEMENTATION_ASSIGNMENTS.md) - Detailed assignment specifications
- [Individual Assignments](../assignments/) - Individual assignment files (A1-A10)

## Usage

Agents should:

1. Review the Quick Start Guide before beginning work
2. Use the Assignment Template for new assignments
3. Update the Progress Tracker when starting/completing assignments
4. Reference individual assignment files in the `assignments/` directory
