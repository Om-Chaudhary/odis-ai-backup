## üìù Adding a New Component

When creating a new component:

1. **Check Component Library First:**
   - See if similar component exists
   - Check for reusable patterns
   - Review design system

2. **Create Component Documentation:**
   - Create `[component-name].md` in this directory
   - Document purpose, props, usage, examples
   - Link to design system

3. **Add to This Index:**
   - Add entry with description
   - Categorize appropriately
   - Link to documentation

4. **Follow Standards:**
   - Use design system from `01-GENERAL/design-system.md`
   - Follow component patterns
   - Ensure accessibility
