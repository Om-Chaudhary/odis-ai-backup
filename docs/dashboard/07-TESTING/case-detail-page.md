# Case Detail Page Testing

Complete testing guide for the case detail page (`/dashboard/cases/[id]`).

## Page Layout

### Test Page Structure

- Navigate to a case detail page
- Document:
  - Page header
  - Breadcrumb navigation
  - Case information display
  - Section organization
  - Scroll behavior
  - Responsive layout

## Case Information

### Test Case Data Display

- Document:
  - Patient name, species, breed
  - Owner name, email, phone
  - Case status badge
  - Case type
  - Created/updated dates
  - Source indicator
  - Field formatting

## Discharge Status Badge

### Test Status Badge

- Document:
  - Badge styling for each status
  - Color coding
  - Icon display
  - Text clarity

## Discharge Timeline

### Test Timeline Component

- Document:
  - Timeline rendering
  - Event entries
  - Date/time formatting
  - Icon usage
  - Status indicators
  - Collapsible sections (if any)

## Call History

### Test Call Display

- Document:
  - Call list items
  - Status indicators
  - Date/time display
  - Audio player component
  - Transcript display
  - Empty states

## SOAP Note Display

### Test SOAP Note

- Document:
  - SOAP note rendering
  - Formatting
  - Sections display
  - Empty states
  - Editing capability (if any)

## Action Buttons

### Test Action Buttons

- Document:
  - Schedule call button
  - Schedule email button
  - Edit case button (if any)
  - Button styling
  - Loading states
  - Success/error feedback
