## Summary of Key Visual Patterns

1. **Color Usage**
   - Brand: Emerald/Teal gradient
   - Status: Color-coded badges
   - Text: Slate gray scale
   - Accents: Appropriate semantic colors

2. **Spacing**
   - Section gap: gap-6 or gap-8
   - Card padding: p-4 or p-6
   - Element spacing: gap-2 or gap-3

3. **Typography**
   - Headers: Bold, tracking-tight
   - Body: Regular, leading-relaxed
   - Labels: Small, font-medium
   - Code: Monospace (GeistMono)

4. **Borders & Shadows**
   - Cards: shadow-xl
   - Borders: border-slate-200/60
   - Accents: border-l-4 border-color
   - Glowing: shadow-[0_0_30px_rgba(...)]

5. **Animations**
   - Duration: 200-300ms standard
   - Easing: ease-in-out
   - Spinning: animate-spin (loaders)
   - Pulsing: animate-pulse (live indicator)
   - Scale: hover:scale-105

6. **Responsive Patterns**
   - Mobile: Single column, stacked
   - Tablet: Wider single column
   - Desktop: Two-column (60/40)
