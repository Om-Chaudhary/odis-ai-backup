# Mobile App Assets

Replace these placeholder files with your actual app assets:

## Required Assets

1. **icon.png** (1024x1024)
   - Main app icon for iOS and Android
   - Should be a square PNG with no transparency (iOS requirement)

2. **splash-icon.png** (512x512)
   - Image displayed on splash screen
   - Should work well on the splash background color (#0d3d38)

3. **adaptive-icon.png** (1024x1024)
   - Android adaptive icon foreground image
   - Should have transparent background
   - Content should be centered within safe zone (66% of image)

4. **favicon.png** (48x48)
   - Web favicon for Expo web builds

## Design Guidelines

- Use the OdisAI brand colors:
  - Primary: #0d3d38 (dark teal)
  - Background: #f2f5f4 (light gray)

- Icon should be recognizable at small sizes
- Avoid fine details that won't render well at small sizes
